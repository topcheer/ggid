Developer Experience

Build in Minutes, Not Days

Clean SDKs in 10 languages. Type-safe. Consistent. Documented.

// Register a user via GGID Go SDK
import "github.com/ggid/ggid/sdk/go/ggid"

func main() {
    client := ggid.New("https://ggid.dev",
        ggid.WithTenantID("00000000-0000-0000-0000-000000000001"),
    )

    // Register a new user
    user, err := client.Register(&ggid.RegisterRequest{
        Username: "alice",
        Email:    "alice@example.com",
        Password: "SecurePass123!",
    })

    // Login and get JWT
    token, err := client.Login("alice", "SecurePass123!")
    fmt.Printf("Token: %s", token.AccessToken)
}
# Register a user
curl -X POST https://ggid.dev/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -H "X-Tenant-ID: 00000000-0000-0000-0000-000000000001" \
  -d '{"username":"alice","email":"alice@example.com","password":"SecurePass123!"}'

# Login and receive JWT
curl -X POST https://ggid.dev/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -H "X-Tenant-ID: 00000000-0000-0000-0000-000000000001" \
  -d '{"username":"alice","password":"SecurePass123!"}'

# Access protected resource
curl https://ggid.dev/api/v1/users \
  -H "Authorization: Bearer <jwt-token>"

});

console.log('JWT:', token.accessToken);

Security & Compliance

Hardened by design with defense-in-depth at every layer.

✓

gRPC TLS Between Services

mTLS with fail-secure fallback for all inter-service communication.

✓

Sanitized Error Responses

Internal errors never leak to clients. All 500 responses are sanitized.

✓

Adaptive Rate Limiting

Per-IP, per-tenant, and per-endpoint rate limits with token bucket algorithm.

✓

Request Body Size Limits

Configurable payload limits to prevent DoS via oversized requests.

✓

Host Header Validation

DNS rebinding protection with configurable allowed hosts.

✓

HSM / KMS Key Management

PKCS#11 integration for hardware-backed JWT signing keys.

✓

PII Obfuscation

Automatic PII redaction in audit logs and SIEM forwarding.

✓

CSRF Protection

Cryptographically secure tokens with SameSite cookie enforcement.

✓

SSRF Prevention

Webhook URL validation with private IP range blocking.

✓

Tamper-Evident Audit

Cryptographic hash chain ensures audit log integrity.

✓

NIS2 / CRA / PIPL

Research-backed compliance frameworks for EU and China.
