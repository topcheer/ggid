# GGID v1.0-stable — Official Release

**Release Date:** July 2026
**Version:** v1.0-stable
**License:** Apache 2.0
**Go Version:** 1.26

## What's New Since v1.0-beta

### Performance
- Login latency reduced 43% (258ms → 148ms) via Argon2id parameter tuning
- PostgreSQL index optimization for hot queries
- Redis caching for list endpoints (30s TTL)

### Security
- JWT 90-day automatic key rotation (KB-329)
- Authorization audit across all POST/PUT/DELETE endpoints
- CORS configuration hardened
- 3 data races found and fixed (atomic.Bool, atomic.Int32, atomic.Value)
- Agent permission scoping framework

### Quality
- `go vet ./...` — 0 warnings
- `go test -race ./...` — 0 data races
- 4461+ test functions (up from ~3000)
- 85+ API security tests (new)
- 33 E2E integration tests (new)
- 11 E2E business flow tests (new)

### UI/UX
- 828 console pages, 822/828 returning 200 (99.3%)
- 0 broken links (404s)
- 5 bugs found and fixed during browser verification
- i18n lazy load framework (15 languages)
- Dark mode, mobile layout, pagination unified
- First-time setup wizard

### Documentation
- 366 user guides (100% feature coverage)
- 292 research documents
- API cookbook (24 curl examples)
- Integration guide (OAuth/SAML/WebAuthn/Client Credentials)
- Admin quickstart, getting started, product overview
- Testing strategy + automation guide
- Security hardening guide

### Deployment
- All-in-one Docker image (8 services in 1 container)
- Docker Compose (full microservices)
- Helm chart + K8s Operator + Terraform
- 8/8 K8s pods Running on amd64
- `--platform linux/amd64` enforced in CI

### OpenAPI
- 704 paths documented (81% of 864 endpoints)
- Swagger UI interactive at `/docs`
- Request/response schema enhancement in progress
