## What is GGID?

GGID is an open-source Identity & Access Management platform built in Go, featuring OAuth 2.1, ReBAC, ITDR, WebAuthn/Passkeys, Zero Trust architecture, and AI Agent Identity — all in a single deployable system.

## Core Capabilities

### Authentication
OAuth 2.1 (PKCE/PAR/JAR/DPoP), OIDC, WebAuthn/FIDO2/Passkeys, SAML 2.0, MFA (TOTP/SMS/push/biometric), Adaptive risk-based auth, Social login, China GM (SM2/SM3/SM4), Password strength engine (zxcvbn), TAP

### Authorization
ReBAC (Zanzibar), ABAC, RBAC with hierarchy, Conditional Access Policies, Continuous Access Evaluation (CAE), Delegation management, PostgreSQL RLS tenant isolation, SoD rules + violation detection

### Security & ITDR
15 MITRE ATT&CK detection rules, Unified risk engine, Hash-chained audit trail (HMAC-SHA256), Continuous Compliance Monitoring (15 controls), DLP egress with PII redaction, SOAR playbooks, Privileged operations audit

### Platform
Multi-tenant, WASM plugins (wazero), Webhook engine (HMAC signed), SCIM 2.0, Feature flags, Secret broker, GraphQL API, i18n (15 languages), First-time setup wizard

## Getting Started

```bash
# 1. Start GGID
docker run -d -p 8080:8080 -p 3000:3000 ggid/ggid-all-in-one:latest

# 2. Login
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -H "X-Tenant-ID: 00000000-0000-0000-0000-000000000001" \
  -d '{"username":"admin","password":"Admin@123456"}'

# 3. Open console
open http://localhost:3000
```
