# GGID v1.0-beta Release Notes

**Release Date:** July 2026
**License:** Apache 2.0
**Go Version:** 1.26

GGID v1.0-beta is the first feature-complete beta of the open-source Identity & Access Management platform. This release delivers enterprise-grade authentication, authorization, identity threat detection, and Zero Trust capabilities in a single Go-native platform.

## Highlights

- **825 console pages** across 8 functional domains
- **864 API endpoints** with 703 documented in OpenAPI (81%)
- **4461 test functions** including 52 API security tests and 33 E2E integration tests
- **364 user guides** + 292 research documents
- **11 SDKs** (Go, React, TypeScript, Java, Python, C#, Rust, Ruby, PHP, Dart, React Native)
- **15 languages** supported in console i18n

## Features

### Authentication
- OAuth 2.1 with PKCE, PAR, JAR, DPoP, Token Exchange (RFC 8693)
- OpenID Connect (Discovery, UserInfo, Back-Channel Logout)
- WebAuthn / FIDO2 / Passkeys with AAGUID allowlist
- SAML 2.0 federation (SP and IdP)
- MFA (TOTP, SMS, push, biometric, WebAuthn)
- Adaptive risk-based authentication (5 signal categories, 20 types)
- Social login (Google, GitHub, Microsoft, Apple)
- China GM compliance (SM2/SM3/SM4)
- Password strength engine (zxcvbn, 5 pattern types, HIBP)
- Temporary Access Pass (TAP)

### Authorization
- ReBAC (Zanzibar-style relationship tuples)
- ABAC (attribute-based conditions)
- RBAC with role hierarchy and inheritance
- Conditional Access Policies (CAP) with session-level CAE
- Delegation management with scoped tokens
- PostgreSQL Row-Level Security (tenant isolation)
- Unified PDP (per-request authorization)
- Separation of Duties (SoD) rules and violation detection
- Privilege creep detection

### Security & ITDR
- 15 MITRE ATT&CK identity threat detection rules
- Unified risk engine with anomaly detection
- Hash-chained audit trail (HMAC-SHA256, tamper-evident)
- Continuous Compliance Monitoring (15 controls, SOC2/SOX/HIPAA/DORA)
- DLP egress with PII redaction
- SOAR playbook integration
- Impossible travel / geo-velocity detection
- Privileged operations audit log (KB-279)
- Break-glass emergency access with full audit
- NHI (Non-Human Identity) risk scoring and behavior analytics

### Platform
- Multi-tenant with per-tenant branding and i18n
- WASM plugin architecture (wazero runtime)
- Webhook engine (HMAC signed, retry, dead-letter)
- SCIM 2.0 inbound/outbound provisioning
- Feature flags system
- Secret broker (zero-trust secret injection)
- GraphQL API layer for identity queries
- First-time setup wizard
- Navigation system with 8 functional domains

### Deployment
- All-in-one Docker image
- Docker Compose (infrastructure + microservices)
- Helm chart (Kubernetes)
- K8s Operator
- Terraform modules
- `.env.example` with 34 configuration variables
- Cloud federation guides (AWS, Azure, Google Cloud)

## Security Fixes

- Fixed `useState(() => {})` crash across 44 console pages
- Fixed React Hooks violations in multiple components
- Fixed DLP endpoint paths (`/auth/dlp/*` → `/identity/dlp/*`)
- Removed all hardcoded credentials from code
- Added nil pool guards in repository layers

## Breaking Changes

None. This is the first public beta.
