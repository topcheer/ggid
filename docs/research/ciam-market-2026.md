# CIAM Market 2026

> Gartner Magic Quadrant analysis and GGID market positioning.

## Gartner MQ for IAM 2025-2026

### Leaders
- **Microsoft Entra ID** — broadest feature set, deep Azure integration
- **Okta** — best-in-class developer experience, largest integration catalog

### Challengers
- **Auth0 (Okta)** — developer-focused, CIAM strength
- **Ping Identity** — enterprise B2B federation

### Visionaries
- **Clerk** — developer-first, fast-growing
- **Ory** — cloud-native open source
- **Stytch** — passwordless-first

### Niche
- **Keycloak** — open source, self-hosted
- **Casdoor** — lightweight, Chinese market

### GGID Positioning

GGID sits in the **open-source challenger** quadrant alongside Ory and Keycloak:

| Strength | Differentiator |
|----------|---------------|
| Multi-tenant RLS | Hard DB isolation (no competitor has this) |
| ABAC + compliance templates | PCI/HIPAA/SOC2 out of the box |
| Audit hash chain | Tamper-evident (unique) |
| AI Agent Identity | No competitor has this |
| SCIM 2.0 | Keycloak lacks this |
| Go performance | Faster than Java/Node alternatives |

## Market Trends 2026

1. **Passwordless everywhere** — passkeys going mainstream (4B+ devices)
2. **AI agent identity** — LLM agents need auth (GGID is early mover)
3. **Identity threat detection** — ISPM becoming table stakes
4. **Decentralized identity** — DIDs, verifiable credentials (early stage)
5. **Post-quantum crypto** — planning phase (NIST FIPS 203/204)
